Trabalhando com Tabelas

Em HTML, as tags são utilizadas para estruturar os conteúdos em uma página web.

Tags Significado
th Quando é utilizado th, indica que é o título da coluna. Significado da sigla: Table Head
td Quando é utilizado td, indica que é o conteúdo da coluna. Significado da sigla: Table Data
tr Quando é utilizado tr, indica que é a linha da tabela. Significado da sigla: Table Row
p A tag P é utilizada quando queremos inserir um parágrafo
h1 A tag h1 é a tag de título em HTML e possui 6 níveis de títulos
a A tag de "a" é a tag âncora, utilizada para criar links, juntamente com o atributo href
img A tag img é para inserir imagem na página web, juntamente com o atributo src
table A tag table é para criar tabela
atributos Em HTML, cada tag pode ter atributos para "formatar" os elementos em HTML
form A tag form é utilizada para criar formulários em HTML
input A tag input é utilizada para criar os campos dos formulários na página web
